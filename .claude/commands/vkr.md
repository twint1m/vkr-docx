# /vkr — Управление ВКР документом

Ты помогаешь оформлять выпускную квалификационную работу (ВКР) в формате .docx по ГОСТ 7.32-2017.

## Python-модуль

```python
from vkr_docx import VKRDocument

vkr = VKRDocument("ВКР 0.2.7.docx")
```

Если пакет не установлен — подскажи пользователю: `pip install -e ~/projects/vkr-docx`

## Поиск файла

Ищи последнюю версию `ВКР *.docx` в текущей директории или в `working_dir` из `config.yaml`.

## Доступные методы

### Настройка
- `vkr.fix_page_setup()` — поля по ГОСТ (настраиваются в config.yaml)
- `vkr.fix_styles()` — стили Heading 1/2/3 и Normal
- `vkr.fix_image_sizes()` — ограничить высоту изображений

### Контент
- `vkr.add_heading1("2 ПРОЕКТИРОВАНИЕ...")` — глава (центр, Bold, новая страница)
- `vkr.add_heading2("2.1 Алгоритм...")` — подраздел (JUSTIFY, Bold, отступ 1.25)
- `vkr.add_heading3("2.1.1 Подпункт")` — пункт
- `vkr.add_body_text("Текст...")` — основной текст (JUSTIFY, отступ 1.25, 14pt)
- `vkr.add_figure(path, "Рисунок N — Описание", width_cm=15)` — рисунок + подпись
- `vkr.add_figure_side_by_side(path1, path2, "Рисунок N — ...")` — два рядом
- `vkr.add_table_caption("Таблица N — Описание")` — подпись таблицы
- `vkr.add_data_table(headers, rows)` — таблица с данными

### Ссылки и навигация
- `vkr.add_crossrefs()` — двусторонние ссылки рисунки↔текст, таблицы↔текст
- `vkr.update_toc()` — обновить содержание

### Версионирование
- `vkr.save()` — сохранить с автоинкрементом правок (B += 1)
- `vkr.save("путь.docx")` — сохранить по указанному пути
- `vkr.save_same()` — перезаписать текущий файл
- `vkr.set_chapter(3)` — переключить на главу 3 → `ВКР 0.3.0`
- `vkr.mark_ready()` — готова → `ВКР 1.0.0`

### CLI
```bash
vkr-docx fix-styles "ВКР 0.2.7.docx"
vkr-docx crossrefs "ВКР 0.2.7.docx"
vkr-docx toc "ВКР 0.2.7.docx"
vkr-docx full-fix "ВКР 0.2.7.docx"
vkr-docx full-fix --overwrite    # перезаписать
```

## Правила ГОСТ

### Страница
A4, поля: лево=3, право=1.5, верх/низ=2 см (настраивается в config.yaml)

### Текст
Times New Roman 14pt, JUSTIFY, отступ 1.25 см, межстрочный 1.5

### Заголовки
- **Heading 1**: 14pt Bold ПРОПИСНЫЕ, по центру, с новой страницы
- **Heading 2**: 14pt Bold, JUSTIFY, отступ 1.25
- **Heading 3**: 14pt Bold, JUSTIFY, отступ 1.25

### Рисунки
- По центру, подпись под рисунком: «Рисунок X — Описание»
- Диаграммы ≤15 см, скриншоты ~8 см, max высота 23 см

### Таблицы
- Подпись НАД таблицей: «Таблица X — Описание»
- Границы, 12pt, заголовки Bold

### Перекрёстные ссылки
- Двусторонние: текст ↔ подпись
- Bookmark names: случайные 12-символьные ID (Google Docs)

### Содержание
- Обновлять после каждой главы
- H1 без отступа, H2 +1 см, H3 +2 см, leader dots

### Версионирование
- `0.A.B` — в работе: A=глава, B=правки
- `1.0.0` — готова, `1.0.N` — правки
