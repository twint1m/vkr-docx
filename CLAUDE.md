## ВКР (выпускная квалификационная работа)

Когда пользователь просит работать с ВКР, дипломом, .docx документом, добавить главу, исправить стили, обновить ссылки/содержание — **используй скилл `/vkr`** и пакет `vkr_docx`.

Ключевые слова: ВКР, диплом, глава, docx, ГОСТ, рисунок, таблица, содержание, перекрёстные ссылки, оформление.

```python
from vkr_docx import VKRDocument

vkr = VKRDocument("ВКР 0.2.7.docx")
```

Последняя версия: ищи файлы `ВКР *.docx` в текущей директории или в `working_dir` из config.yaml.
